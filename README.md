# To_Do_Skillcraft
A lightweight, dependency-free to-do app for organizing tasks into custom lists. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

## Features

- Create multiple color-coded task lists
- Add tasks with due date and time
- Mark tasks complete
- Edit tasks in place
- Delete tasks
- Automatic overdue flagging
- Data saved locally in your browser (`localStorage`) — no login or server needed

## File Structure

```
todo-app/
├── index.html   → page structure
├── style.css    → styling
├── script.js    → app logic
└── README.md
```

Keep all three files in the same folder — `index.html` links to the other two.

## How to Run

Open `index.html` in any modern browser. No installation or internet connection required.

## Notes

- Data is stored per-browser/device only (via `localStorage`), not synced across devices.
- Clearing browser data will erase saved tasks.
